**[Pint](http://sblisesivdin.github.io/pint)** Pint is a powerful and user-friendly UI/GUI tool for conducting Density Functional Theory (DFT) and molecular dynamics (MD) calculations. In near future, Machine Learning featuress will be added.

`gpaw-tools` is transitioning to `Pint`.

Dear Users, 
When it first started, `gpaw-tools` was based on GPAW, but today it uses many Python libraries such as ASE, Elastic, Phonopy, ASAP3, OpenKIM, and others. In the near future, it will be enriched with many Python libraries such as SevenNet, CHGNet, and MACE-torch. At this point, we have embarked on a new naming convention to better define the software. After this stage, it will be called `Pint`, which stands for Pythonic Interface for Nanoscale Tasks. The `gpaw-tools` project has moved to its new home to achieve a more comprehensive, modern, and ML-supported structure.

What does this mean for you?

gpaw-tools (v25.x): Has been placed in maintenance mode. No new features will be added except for critical bug fixes.

Pint (v26.x): Includes all the capabilities of gpaw-tools (excluding gg.py), but offers them as a modern Python package (pip install). It also incorporates modern machine learning capabilities such as MACE and CHGNet.

**IMPORTANT INFORMATION: Pint is in a beta phase. Please continue to use gpaw-tools until further notice. You can view the development at [Release Notes](https://github.com/sblisesivdin/pint/blob/main/RELEASE_NOTES.md)**
